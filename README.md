# Yomis Calculator

**A scientific calculator built with vanilla HTML, CSS, and JavaScript**  

[Live Demo](https://yomiscalculator.netlify.app/)

---

## 📌 Project Overview
**Yomis Calculator** is a **scientific calculator application** implemented entirely with plain HTML, CSS, and JavaScript.  
It demonstrates DOM manipulation, event handling, styling, and client-side logic without any frameworks or libraries.  

This project showcases your ability to create **functional web tools** from scratch, with a responsive UI and interactive experience.  

---

## 📊 Key Features

### 🔢 Basic Arithmetic
- Addition (+), subtraction (−), multiplication (×), division (÷)
- Instant calculations with the equals (`=`) button

### 📐 Scientific Functions
- Advanced math functions such as square root (√), power (^), percentage (%)
- Optional: trigonometric or other scientific operations (if implemented)

### 🧠 Vanilla JavaScript Logic
- Handles all calculator operations with pure JS
- Updates the display dynamically
- Maintains calculation state and logic entirely client-side

### 🎨 Responsive UI
- Styled using CSS
- Traditional calculator layout that adapts to different screen sizes

### 📱 Interactive Experience
- Real-time display updates when buttons are clicked
- Clear (`C`) and delete (`←`) buttons to manage input
